# Theme Contract

Themes control layout, markup presentation, typography, spacing, decorative styles, responsive behavior, and theme-specific components.

Shared core logic controls content loading, schema validation, draft filtering, scheduled publishing, sorting, pagination, tag generation, related content, URL generation, SEO, RSS, sitemap, search metadata, Mermaid detection, and reading time.

Required CSS tokens:

```css
--color-background
--color-surface
--color-surface-muted
--color-text
--color-text-muted
--color-border
--color-accent
--color-accent-contrast
--font-body
--font-heading
--font-code
--content-width
--article-width
--radius-small
--radius-medium
```
